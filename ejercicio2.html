Que es html

HTML es un lenguaje de marcado que nos permite hacer la estructura de una página web.

Que programas usan html

Atom, Visual Studio Code, Sublime Text, Brackets, Notepad++

Historia de HTML

HTML (HyperText Markup Language) es el lenguaje fundamental para estructurar y presentar contenido en la web. Su historia comienza en 1989, cuando Tim Berners-Lee, físico del CERN, desarrolló un sistema de hipertexto para facilitar el intercambio de información entre científicos.
